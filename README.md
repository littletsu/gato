Project structure:
* /assets
    - general assets for the game
* /engine
    - game engine and modules that complement love2d
* /entity
    - game objects that are not pooled
* /objects
    - game objects that are pooled
* /scenes
    - game scenes/levels
* /utils
    - useful 3rd party libs
* conf.lua
    - love2d config file
* main.lua
    - game entry point
* run.bat
    - runs the game with lovec
